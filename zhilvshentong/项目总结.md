# 智旅深瞳项目总结

## 项目概述

智旅深瞳是一款基于React和TypeScript的智能旅游助手Web应用，提供智能行程规划、实时信息聚合、文化体验推荐和碳足迹计算等功能。项目采用现代化的前端技术栈，包括React 18、TypeScript、Ant Design、React Router和Vite构建工具。

## 如何运行项目

### 环境要求
- Node.js 16.0 或更高版本
- npm 7.0 或更高版本

### 安装和运行步骤
1. 安装Node.js（如果尚未安装）
   - 访问 [Node.js官网](https://nodejs.org/) 下载并安装
   - 验证安装：`node --version` 和 `npm --version`

2. 安装项目依赖
   ```bash
   cd zhilvshentong
   npm install
   ```

3. 启动开发服务器
   ```bash
   npm run dev
   ```

4. 在浏览器中访问应用
   - 默认地址：http://localhost:3000

5. 构建生产版本（可选）
   ```bash
   npm run build
   ```
